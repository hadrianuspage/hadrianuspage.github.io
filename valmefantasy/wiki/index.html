ValmeFantasy Custom Enchant Commands

You can finding a new enchantments by on enchanting table, by explored or chest

/eenchants list : to see enchantment lists and can used for yourself or another players
/eenchants disenchant : to disenchant enchantments

ValmeFantasy MMO Commands

/sk help : Shows the command help page.
/skills : open the skills menu
/stats : open the stats menu
/mana : Shows your current mana.
/sk claimitems : open the menu to claim items
/sk sources : Opens the sources menu for a skill, which displays the ways to gain XP

ValmeFantasy AutoPickup Commands

By default has been enabled. So you can use this command to toggle

/auto : toggle auto pickup to on or off
/autodrops : toggle autodrops mobs to on or off
/autosmelt : toggle autosmelt ores to on or off

ValmeFantasy Trade Commands

/axtrade {player} : trade to another player
/axtrade accept {player} : accept trade
/axtrade deny {player} : denied trade
/axtrade toggle : toggle your trade to make it enabled or disabled

ValmeFantasy Storage Player Commands

/axpv : open chest virtual storage
/axpv {number} : choose chest virtual storage you want

ValmeFantasy Skin Customization Commands

You can see all skin commands by using /skin command

/skin {username} : Ex /skin Notch
/skin clear : removed your skin
/skin random : randomized your skin from username choosed
/skin set {username} : allowed you to set your skin so if you leave the server and the skin never changed. Ex /skin set Notch or using url
/skin url {url} : allowed you to set your skin from url. Ex /skin url https://i.imgur.com/9igpkh6.png. You must using imgur.com to using skin url
