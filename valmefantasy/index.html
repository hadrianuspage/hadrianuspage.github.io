📚 Tutorial & Guide

Panduan Lengkap Bermain di Valme Fantasy

🎮 Tutorial Bedrock Edition

Cara Join Server Bedrock:

  1. Download dan install Minecraft Bedrock Edition
  2. Buka game dan pilih "Play"
  3. Pilih tab "Servers"
  4. Scroll ke bawah dan pilih "Add Server"
  5. Isi form dengan data berikut:
    • Server Name: Valme Fantasy
    • Server Address: 147.185.221.26
    • Port: 5826
  6. Klik "Save" dan pilih server untuk join

☕ Tutorial Java Edition

Cara Join Server Java:

  1. Download dan install Minecraft Java Edition
  2. Buka game dan pilih "Multiplayer"
  3. Klik "Add Server"
  4. Isi form dengan data berikut:
    • Server Name: Valme Fantasy Java
    • Server Address: 147.185.221.19:25783
  5. Klik "Done" dan double-click server untuk join
